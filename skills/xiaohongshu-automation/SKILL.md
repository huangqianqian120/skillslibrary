# 小红书自动化助手 🏛️

监控博物馆/文博/文物相关内容，自动生成并发布笔记。

## 核心功能

### 1️⃣ 关键词监控
- 搜索：小红书笔记、热门话题
- 追踪：点赞量、评论趋势
- 灵感：发现爆款内容规律

### 2️⃣ AI 内容生成
- 根据关键词生成文案
- 适配小红书风格
- 生成标题+正文+标签

### 3️⃣ 自动发布（待配置）
- 浏览器自动化登录
- 自动上传图片+发布

## 使用方式

### 日常监控
```
搜索博物馆相关内容
看看最近文博有什么热门
文物相关的笔记有哪些
```

### 生成内容
```
帮我写一篇博物馆笔记
生成一个关于文物的文案
写个小红书风格的帖子
```

### 发布（需要配置）
```
发布这篇笔记
上传并发布
```

## 内容风格

**博物馆/文博方向：**
- 🏛️ 文物背后的故事
- 📜 历史冷知识
- 🎨 艺术鉴赏
- 🗺️ 博物馆打卡攻略

## 定时任务（可选）

每天/每周自动执行：
- 关键词搜索
- 热门内容摘要
- 生成新笔记草稿

## 注意事项

⚠️ 自动化发布需要：
- 账号 cookies 认证
- 遵守小红书社区规范
- 适度使用，避免封号
