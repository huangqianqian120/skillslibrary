# Skills CLI

Command line interface for Skills Library.

## Description

A CLI tool that lets you browse, search, and install AI Agent skills directly from your terminal.

## Features

- List all available skills
- Search skills by name
- View skill details
- Install skills to your project

## Installation

```bash
npm install -g skills-cli
```

## Usage

```bash
# List all skills
skills list

# Search for a skill
skills search copywriting

# View skill details
skills cat copywriting

# Install a skill
skills install copywriting
```
