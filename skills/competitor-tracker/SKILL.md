# Competitor Tracker

Track competitor activities and updates.

## Tracking Targets
- Competitor news
- Product launches
- Pricing changes
- Feature updates
- Social media activity
- Hiring/expansion

## Usage

### Add Competitor
```
Track [company name]
Monitor [competitor] news
```

### Get Updates
```
What's new with [competitor]?
Recent updates from [company]
Competitor activity report
```

### Competitive Analysis
```
Compare [company A] vs [company B]
[List competitors] update summary
```

## Output Format
- Recent activities (last 7-30 days)
- Key updates with impact analysis
- Links to sources
