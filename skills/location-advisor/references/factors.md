# 选址因素库

## 五大维度

### 1. 交通便利性 (默认权重25%)
- 地铁站距离 (10%)
- 公交线路数 (5%)
- 停车位数量 (5%)
- 物流配送距离 (5%)

### 2. 成本效益 (默认权重25%)
- 租金 (10%)
- 装修成本 (5%)
- 人力成本 (5%)
- 运营成本 (5%)

### 3. 市场潜力 (默认权重25%)
- 3km人口密度 (10%)
- 人均消费能力 (5%)
- 日均客流量 (5%)
- 区域增长趋势 (5%)

### 4. 竞争环境 (默认权重15%)
- 竞争对手数量 (5%)
- 竞争对手距离 (5%)
- 市场集中度 (3%)
- 差异化空间 (2%)

### 5. 政策风险 (默认权重10%)
- zoning规划 (4%)
- 证照难度 (3%)
- 拆迁风险 (2%)
- 政策补贴 (1%)
