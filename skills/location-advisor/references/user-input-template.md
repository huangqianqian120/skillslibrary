## ⚖️ STEP 3: 权重偏好

请选择或调整各因素重要性：

### 快速选择（任选其一）：
- [ ] 流量优先：交通30% + 市场30% + 成本15%
- [ ] 成本优先：成本40% + 交通20% + 市场20%
- [ ] 平衡型：各20%

### 精细调整（0-100%）：

| 因素 | 你的权重 | 说明 |
|------|----------|------|
| 交通便利性 | % | 地铁、公交、停车 |
| 成本效益 | % | 租金、人力、运营 |
| 市场潜力 | % | 人口、消费、客流 |
| 竞争环境 | % | 竞品数量、差异化 |
| 政策风险 | % | zoning、证照、拆迁 |
| **合计** | 0% | 需要=100% |
