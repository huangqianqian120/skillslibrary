# AI News Aggregator

Aggregate AI/LLM/Agent news from multiple sources.

## Sources
- Hacker News (AI, LLM, Agent tags)
- GitHub Trending (AI, agent frameworks)
- arXiv (AI papers)
- Hugging Face (model releases)

## Usage

### Daily Brief
```
Get today's AI news summary
```

### Specific Topic
```
GitHub trending AI projects
Latest LLM releases
Recent AI papers on agents
```

### Search
```
Search for [topic] news
Find news about [specific technology]
```

## Output Format
- Top 5-10 important news
- Each with: title, key points, link
- Categorized by topic
