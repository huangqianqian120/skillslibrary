# Skills CLI

CLI tool for Skills Library - Browse and install AI Agent skills from command line.

## Install

```bash
npm install -g skills-cli
```

## Usage

```bash
# List all skills
skills list

# Search skills
skills search copywriting

# Show skill details
skills cat copywriting

# Install skill
skills install copywriting
```
